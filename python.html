    高数
  1. 求不定积分：∫x3(lnx)2dx
    sympy指令： 
    from sympy import *
x = symbols('x')
integrate(x * sin(2 * x), x)
    结果：-x*cos(2*x)/2 + sin(2*x)/4
  2. 求不定积分：∫sec3xdx
    sympy指令： 
    from sympy import *
x = symbols('x')
integrate(sec(x) ** 3, x)
    结果：-log(sin(x) - 1)/4 + log(sin(x) + 1)/4 - sin(x)/(2*sin(x)**2 - 2)
    与手算得到的结果形式上有不同，可能是表示不一样
    线代
  1. 计算下列乘积： 
    | 4  3  1 | | 7 |
| 1 -2  3 | | 2 |
| 5  7  0 | | 1 |
    sympy指令： 
    from numpy import *
a = array([[4, 3 ,1], [1, -2, 3], [5, 7, 0]])
b = array([7, 2, 1]).reshape(3,1)
a @ b
    结果： 
    array([[35],
       [ 6],
       [49]])
  2. 计算下列乘积： 
                   | 1  3  1 |
| 2  1  4  0 | | 0 -1  2 |
| 1 -1  3  4 | | 1 -3  1 |
               | 4  0 -2 |
    sympy指令： 
    from numpy import *
a = array([[2, 1, 4, 0], [1, -1, 3, 4]])
b = array([[1, 3, 1], [0, -1, 2], [1, -3, 1], [4, 0, -2]])
a @ b
    结果： 
    array([[ 6, -7,  8],
       [20, -5, -6]])
